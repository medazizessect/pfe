\chapter{Implémentation du Prototype et Expérimentations}

\section{Introduction}
Cette étape couvre la concrétisation technique : réalisation par sprints, choix technos, assemblage back/front IA et expérimentation terrain.

\section{Choix technologiques et architecture}
L’ensemble du système repose sur :
\begin{itemize}
  \item \textbf{Backend IA} : Python (modules Python YOLOv8, OpenCV pour traitement image, FastAPI pour REST API)
  \item \textbf{Interface Mobile} : React Native (expo)
  \item \textbf{Front web admin} : ReactJS
  \item \textbf{Base de données} : PostgreSQL (hébergement sur cloud)
  \item \textbf{Stockage images/vidéos} : AWS S3
  \item \textbf{Collaboration} : GitHub, Notion, Trello
\end{itemize}
