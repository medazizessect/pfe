\chapter{Préparation, Planification et Architecture Système}

\section{Introduction}
Le développement d'une plateforme d'intelligence artificielle appliquée à la mobilité urbaine nécessite une rigueur organisationnelle et technique sans faille. Ce chapitre décrit les étapes préparatoires, la méthodologie de gestion de projet adoptée, le découpage des tâches en sprints, ainsi que l'architecture système cible et les diagrammes de conception (UML).

\section{Méthodologie de Gestion de Projet : Hybridation Scrum et XP}
Pour répondre à l'incertitude inhérente aux projets d'intelligence artificielle tout en garantissant un code robuste, nous avons opté pour une approche hybride combinant Scrum et Extreme Programming (XP).

\subsection{Le cadre Scrum : Pilotage et Alignement Métier}
Scrum a été utilisé pour structurer le temps et maximiser la valeur livrée aux parties prenantes.
\begin{itemize}
    \item \textbf{Sprints} : Un rythme itératif permettant de réajuster rapidement les paramètres.
    \item \textbf{Cérémonies} : Sprint Planning, Daily Scrum, et Sprint Review pour la validation des modules.
\end{itemize}

\subsection{Extreme Programming (XP) : Excellence Technique}
\begin{itemize}
    \item \textbf{Intégration Continue (CI)} : Chaque modification est packagée via Docker.
    \item \textbf{Conception Simple} : Architecture découplée entre `capture_reconnaissance` et `bi_analytics`.
    \item \textbf{Standards de codage} : Typage strict sous Python 3.11 avec Pydantic.
\end{itemize}

\section{Architecture d'Acquisition : Du PoC à la Réalité Municipale}
Dans le cadre d'un déploiement réel en conditions de "Smart City", le système \textit{Circuli} est conçu pour s'interfacer directement avec le Système de Gestion Vidéo (VMS) de la municipalité (caméras IP CCTV via protocole RTSP). 

Toutefois, durant la phase de développement académique, l'accès direct aux serveurs sécurisés des municipalités est restreint. Pour surmonter cette contrainte, nous avons mis en place un \textbf{simulateur urbain} utilisant \texttt{yt-dlp} pour extraire les flux en direct de caméras urbaines diffusées sur YouTube, validant ainsi la chaîne d'ingestion algorithmique de manière iso-fonctionnelle.
